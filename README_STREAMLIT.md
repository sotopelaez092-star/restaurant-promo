# Restaurant Promo Streamlit 应用

## 本地运行

1. 安装 Python 依赖：
```bash
pip install -r requirements.txt
```

2. 启动 Streamlit 应用：
```bash
streamlit run app.py
```

3. 在浏览器中打开显示的地址（通常是 http://localhost:8501）

## 部署到 Streamlit Cloud

1. 将代码推送到 GitHub 仓库

2. 访问 [Streamlit Cloud](https://streamlit.io/cloud)

3. 连接你的 GitHub 仓库

4. 选择 `app.py` 作为主文件

5. 点击部署

## 注意事项

- 确保项目中已安装 Node.js 和 Remotion
- Streamlit Cloud 部署时需要配置 Node.js 环境
- 视频渲染可能需要较长时间，建议本地测试后再部署
